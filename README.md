# Motivational-Quotes-Extension
  The Motivational Quotes Chrome Extension is a simple yet powerful tool designed to provide daily inspiration and encouragement to users. With just a click, users can access a curated collection of motivational quotes from a variety of sources.

Key Features:
  Random Quote Generation: Get a new motivational quote every time you open the extension.
  Easy to Use: Simply click the "New Quote" button to refresh the quote.
  Beautiful Design: Enjoy a sleek and modern user interface with smooth transitions and elegant typography.
  Personal Growth: Start your day with a positive mindset and stay motivated throughout the day.

Technologies Used:
  HTML
  CSS
  JavaScript
